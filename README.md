# TP 7 React Native - Architecture & Offline

Ce projet contient l'implémentation du TP 7.

## Fonctionnalités
- **API Fetching**: Récupération de tâches depuis `jsonplaceholder` via `axios` ou `fetch`.
- **Mode Hors Ligne (offline)**:
  - Base de données SQLite locale.
  - Ajout, Modification, Suppression de tâches hors ligne.
  - Persistance des données.
- **Thème Sombre/Clair**: Gestion du thème via `Context` et persistance avec `AsyncStorage`.

## Installation
1. `npm install`
2. `npx expo start`
